# Security Policy

## Reporting a Vulnerability

このプロジェクトでセキュリティ上の脆弱性を発見した場合は、GitHub Issues ではなく、以下の方法で非公開にご報告ください。

**報告先**: [GitHub Security Advisories](https://github.com/rymetry/verdict/security/advisories/new)

報告に含めていただきたい情報:

- 脆弱性の説明
- 再現手順
- 影響範囲の評価

報告後、できる限り速やかに対応いたします。

## Scope

対象となるセキュリティ問題:

- XSS（クロスサイトスクリプティング）
- 依存パッケージの脆弱性
- 意図しない情報漏洩
