# Contributor Covenant Code of Conduct

## Our Pledge

私たちはメンバー、コントリビューター、リーダーとして、年齢、体格、障害の有無、民族性、性的特徴、性自認・性表現、経験レベル、学歴、社会経済的地位、国籍、容姿、人種、宗教、性的指向にかかわらず、誰もがハラスメントのないコミュニティに参加できることを誓います。

## Our Standards

ポジティブな環境づくりに貢献する行動:

- 歓迎的で包括的な言葉遣い
- 異なる視点や経験の尊重
- 建設的なフィードバックの受け入れ
- コミュニティにとって最善なことへの注力
- 他のコミュニティメンバーへの共感

許容されない行動:

- 性的な言葉や画像の使用、および不適切な性的関心やアプローチ
- 荒らし、侮辱的/軽蔑的なコメント、個人的または政治的攻撃
- 公的または私的なハラスメント
- 明示的な許可なく他者の個人情報を公開すること
- 専門的な場で不適切と合理的に判断されるその他の行為

## Enforcement

不適切な行動があった場合は、GitHub Issues またはリポジトリオーナーに報告してください。すべての報告は迅速かつ公平に対応されます。
